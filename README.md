## Asp-Demo 開發環境安裝說明

#### 本機開發環境需搭配使用相關第三方服務，採用Docker方式安裝，依照下列程序安裝

* 執行docker-compose啟動相關第三方服務
  ```bash
  cd .\asp-demo-docker ; docker-compose up -d
  ```
